📁 Project Overview

CTrace GUI is an Electron-based desktop application that provides a modern, VS Code-like interface for running CTrace static analysis on C/C++ code.

🏗️ Architecture

  • Main Process: Handles file system operations, IPC communication, and system dialogs
  • Renderer Process: Manages the UI, editor, and user interactions
  • Manager Pattern: Modular architecture with specialized managers for different concerns

🚀 Getting Started

📋 Prerequisites

  • Node.js (v18+)
  • npm
  • Electron
  • CTrace binary

⚡ Quick Commands

  • npm install - Install dependencies
  • npm start - Run the application
  • npm run dist - Build for distribution
  • npm run docs - Generate documentation

🎨 Key Features

  • 📁 File Management: Open individual files or entire workspaces
  • 🌳 File Tree Explorer: Navigate project structure with refresh and auto-watch
  • 💻 Code Editor: Syntax-highlighted editor with line numbers
  • 📑 Tab Management: Multi-file editing with tab interface
  • 🔧 CTrace Integration: Run static analysis directly from GUI
  • 🔍 Global Search: Search across workspace files
  • 🔔 Notifications: User-friendly notification system
  • ⚡ File Watching: Automatic file tree updates on changes
